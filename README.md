### AIKIDO

#### Prerequisites

You need `node`, `npm` to be installed on your machine. That's all.

#### Installation

```bash
git clone https://github.com/jbehuet/aikido.git   
cd aikido
npm i
```

#### Development

`npm start` serves the app on http://localhost:3000

#### Build

`npm run build`

#### Deploy (to firebase)

`firebase deploy`
